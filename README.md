# Darwin · Personal Portfolio

A personal portfolio site built on the Strata template (by HTML5 UP) with custom overrides:

- Vertical layout with a frosted-glass sticky navigation bar
- Vanta.js Birds background in the header (sunset-gold / dusk-plum palette)
- Cream / beige color scheme throughout
- Smooth-scroll anchors for Home / About / Portfolio / Contact

## Local preview

Any static file server works. For example with Python:

```bash
python -m http.server 8765
```

Then open <http://localhost:8765>.

## Deployment

Hosted on GitHub Pages at `https://<username>.github.io`.
